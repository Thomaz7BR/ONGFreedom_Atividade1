# ONG Freedom - Atividade 1

Projeto referente à Entrega I da disciplina Fundamentos da Web e Estruturação de Interfaces.

## Descrição
Este projeto contém as páginas HTML semânticas da ONG Freedom:
- Página inicial (index.html) com informações da ONG e contato.
- Página de projetos (projetos.html) com voluntariado e doações.
- Página de cadastro (cadastro.html) com formulário completo e máscaras de input.
